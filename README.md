# Sale Project CRUD
Proyecto final para el curso de Programación Orientada a Objetos en Java, 
este proyecto está trabajado con Maven y el look and Feel de 
Flatlaf https://www.formdev.com/flatlaf/

## Eventos
Actualmente, este proyecto sigue en desarrollo para mejorar sus funcionalidades, por ahora solo funcionan los botones de relacionados con los formularios "Cajero y Productos". Cada vez que haya una actualización estaré subiendo el proyecto al Google Drive donde está alojado para aquellos que quieran el proyecto completo.

## Versión de JDK y Software
El Software usando fue 
- JDK 26.0.1
- IntelliJ IDE
- Apache NetBeans IDE 30
- JDBC 8.3.0
- Flatlaf 3.2.7
- MySQL

## Probar el Proyecto en Local
Inicia un CMD e ingresa el siguiente comando:
